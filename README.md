## Customization

### To Update Contact Information
Edit the contact section in `index.html`:
```html
<p><a href="tel:08010024423">080100 24423</a></p>
```

### To Change Colors
Modify CSS variables in `styles.css`:
```css
:root {
    --primary-color: #ff4444;
    --secondary-color: #222222;
    --accent-color: #ffcc00;
    /* ... */
}
```

### To Add More Reviews
Add review cards in the reviews section:
```html
<div class="review-card">
    <div class="stars">★★★★★</div>
    <p>"Your review text here"</p>
    <p class="reviewer">- Member Name</p>
</div>
```
